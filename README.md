# AI Hotel

Static GitHub Pages landing page for `aihotel.rest`.

## Local preview

Open `index.html` directly in a browser, or serve the folder with any static server:

```sh
python3 -m http.server 8080
```

Then visit `http://localhost:8080`.

## GitHub Pages

The `CNAME` file is set to `aihotel.rest`. In GitHub, enable Pages from the repository root on the `main` branch, then point DNS for the domain at GitHub Pages.
